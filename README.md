# KACHY DHAGGY

A single-page website concept for a luxury fashion house, built with vanilla HTML, CSS, and JavaScript — no frameworks or dependencies.

## Live Preview
Live preview available on https://raibayub.github.io/kachy-dhaggy/

## Features

- Dark / light mode with saved preference
- Scroll-reveal animations
- Animated stat counters
- Fully responsive layout with mobile navigation
- Sections: Hero, Collections, Best Sellers, Brand Story, Testimonials, Contact

## Tech Stack

HTML5 · CSS3 · Vanilla JavaScript

## Getting Started

```bash
git clone https://github.com/raibayub/kachy-dhaggy.git
cd kachy-dhaggy
```

Open `index.html` in your browser, or serve it locally:

```bash
npx serve .
```
